# 64bit Tap

## How do I install these formulae?

`brew install 64bit/tap/<formula>`

Or `brew tap 64bit/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "64bit/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
